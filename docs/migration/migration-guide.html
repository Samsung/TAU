Migration Guide

Tizen 2.3 has been a major step forward for the mobile platform. In this release the UI framework has been written from ground up keeping performance in the first place.

We provide this migration guide for all developers which want to move their apps, designed to work with Web UI Framework, forward and become compatible with Tizen's new UI Framework - TAU.
